A set of custom goal packs for the BingoSync Hollow Knight mod.

To use, place the downloaded file in the CustomProfiles folder for the BingoSync mod settings, found in the Hollow Knight saves folder. 
When generating a board, make sure to choose your chosen pack as the profile. Everything else is done as normal.

Each goal pack (with the exception of Wingy) requires both BingoGoalPack1 and BingoGoalPack2.

Each of the goal packs includes:

Bosses: Has every boss in the game.

Dream Nail: Contains exclusively goals requiring Dream Nail.

Everything: Has every goal from every goal pack (vanilla, extended, extended+, item rando, hall of gods, grubs, relics, hard saves, and BingoGoalPack2).

Fast Pack: Contains exclusively goals considered "fast."

Goal Pack 1 + 2: Contains all goals from the vanilla goal pack and BingoGoalPack2.

Goal Pack 2: Contains all goals in BingoGoalPack2, but no others.

Terrible: Contains exclusively goals that are really long.

Wingy: A goal pack made specifically for wingy, removing impossible goals for the format, as well as a few item rando goals. Credits to Blossom, Aggy, and spleen for helping make the pack.
